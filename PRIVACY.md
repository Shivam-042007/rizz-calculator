# Privacy Policy & Disclaimer

**Last Updated:** February 2026

The **Rizz Calculator** ("the App") is an experimental, open-source web application built for entertainment purposes. We respect your privacy and are committed to transparent data practices. 

## 🛡️ Data Collection & Privacy (Zero Retention)

This application requires users to upload a facial image (selfie) to function. **We do not save, store, or sell your biometric data.**

1. **Local Processing:** Image uploads are initially converted to Base64 format directly within your web browser.
2. **API Transmission:** The Base64 string is sent securely to the **Google Gemini API** for algorithmic analysis.
3. **Zero Storage:** Neither the App's hosting server (Netlify) nor the database retains these images. Once the session ends or the page is refreshed, the image data is permanently destroyed.

## 🤖 Third-Party Services

To provide the AI analysis, this app relies on third-party infrastructure:
* **Google Generative AI (Gemini):** Used to process the uploaded image and generate the JSON response. By using this App, your image is temporarily processed by Google's servers. You can review Google's API Privacy Policy [here](https://policies.google.com/privacy).
* **Netlify / GitHub Pages:** Used for hosting the static front-end application. 
* **Analytics:** We may use basic, anonymized web traffic monitors (like Google Analytics) to track page visits and feature usage (e.g., how many times the "Export Report" button is clicked). This data does not track individual identities or link to uploaded images.

## ⚠️ Official Disclaimer

**This application is a novelty generator intended strictly for comedic and entertainment purposes.**

* **Not Real Advice:** The "Aura", "Rizz", and "Looksmaxxing" evaluations are generated by an AI instructed to use exaggerated Gen-Z internet slang. 
* **No Medical or Psychological Value:** The App's analysis of facial architecture, stress, or symmetry is completely algorithmic and fictional. It does not replace, nor does it attempt to provide, real medical, dermatological, or psychological evaluations.
* **Use Responsibly:** Please do not take the results seriously. The AI is designed to playfully roast or hype up the user's "vibes" based on a single, static image.
