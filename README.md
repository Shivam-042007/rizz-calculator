**RIZZ CALC** is a futuristic, cyberpunk-themed web application that leverages **Google’s Gemini 2.5 Flash AI** to perform deep biometric and "aura" analysis. Built with a heavy focus on Gen-Z aesthetics, it provides a playful yet detailed "looksmaxxing" report based on facial architecture.

### 🔗 Try it Live: [hype-rizz.netlify.app](https://hype-rizz.netlify.app)

---

## ✨ Key Features

* **Biometric Scanning HUD**: A high-tech, horizontal glassmorphism dashboard that mimics a military-grade scanner.
* **AI Aura Analysis**: Generates a total Rizz Score, Percentile, and assigned "Rizz Tier" (e.g., GigaChad, Unspoken Rizz, or Cooked).
* **Deep Feature Breakdown**: Analyzes 8 specific facial attributes including Jawline, Eyes, and Confidence, providing custom advice for each.
* **Dynamic Report Generation**: Users can download a custom-styled `.jpg` report card to save their stats.
* **Native Sharing**: Integrated Web Share API to easily blast your aura score to your friends.

---

## 🚀 Tech Stack

* **Frontend**: React.js with Vite for lightning-fast performance.
* **Styling**: Tailwind CSS for custom glassmorphism and neon HUD effects.
* **AI Engine**: `@google/generative-ai` SDK (Gemini 2.5 Flash).
* **Icons**: Lucide React for crisp, futuristic UI elements.
* **Deployment**: Hosted on **Netlify** with GitHub automation.

## 🛠️ Installation & Setup

1.  **Clone the Repo**
    ```bash
    git clone [https://github.com/Shivam-042007/rizz-calculator.git](https://github.com/Shivam-042007/rizz-calculator.git)
    cd rizz-calculator
    ```

2.  **Install Dependencies**
    ```bash
    npm install
    ```

3.  **Setup Environment Variables**
    Create a `.env` file in the root directory and add your Google AI Studio API Key:
    ```env
    VITE_GEMINI_API_KEY=your_api_key_here
    ```

4.  **Run Development Server**
    ```bash
    npm run dev
    ```
